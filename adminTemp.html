Buttons color variation

Button class="" Description
x_btn 중요도가 높지 않은 과업에 사용하는 기본 회색 버튼.
Standard gray button with gradient.
x_btn x_btn-inverse 중요도가 높지 않은 과업에 사용하는 기본 검정 버튼.
디자인 맥락에 따라 기본 회색 버튼 대신 사용할 수 있다.
Alternate dark gray button, not tied to a semantic action or use.
x_btn x_btn-info 양자 택일(취소|확인)을 해야 하는 경우 기본 회색 버튼과 함께 사용.
[취소]는 기본 회색 버튼, [확인]은 색이 들어있는 이 버튼을 사용하면 적합.
Used as an alternative to the default styles.
x_btn x_btn-primary 중요도가 높은 과업에 사용하는 버튼. 보통 폼을 서밋하는 경우에 사용.
Provides extra visual weight and identifies the primary action in a set of buttons.
x_btn x_btn-success 과업 성공을 의미한다. 과업 이후의 페이지로 연결하는 버튼.
Indicates a successful or positive action.
x_btn x_btn-warning 이 버튼을 누를 때 신중해야 한다.
Indicates caution should be taken with this action.
x_btn x_btn-danger 이 버튼을 누를 때 데이터베이스에 저장된 내용을 지운다.
Indicates a dangerous or potentially negative action.
x_btn x_btn-link button 요소가 link처럼 보이기를 원할 때 사용.
Deemphasize a button by making it look like a link while maintaining button behavior.
x_btn x_disabled button 요소 비활성.
Button element disabled.
